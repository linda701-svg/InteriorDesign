Project Title: Interior Design Website

Reference Design:
https://kits.roxthemes.com/archevo/template-kit/home/

Goal

Design and develop a modern, responsive Interior Design Website that showcases interior projects, services, and design concepts, allowing users to explore designs and enabling admins to manage content through a dashboard.

Tech Stack

Frontend: React, CSS, Bootstrap

Backend: Node.js, Express.js

Database: MongoDB (Mongoose)

Authentication: JWT-based authentication

Image Upload: Multer (local or cloud storage)

No AI, No Tailwind CSS, No TypeScript, No use vite

Core Features
1. User Interface (Frontend)

Home page with hero banner & featured designs

About Us & Services pages

Interior project gallery (residential, commercial, office, etc.)

Project details page with images and descriptions

Contact form with backend integration

Responsive layout using Bootstrap & custom CSS

2. Admin Dashboard

Secure admin login

Dashboard summary cards:

Total Projects

Total Categories

Total Inquiries

Manage interior projects (CRUD)

Upload & manage project images

Manage project categories (Living Room, Bedroom, Office, Kitchen, etc.)

View and respond to user inquiries

3. Backend API (Express + MongoDB)

RESTful APIs for:

Projects

Categories

Contact inquiries

Admin authentication

Image upload & storage

Input validation & error handling

Pagination for project listings

4. Database Structure (MongoDB)

Projects Collection

Title

Description

Category

Images

Status (Active / Hidden)

Created date

Categories Collection

Description

Users (Admin)

Email

Password (hashed)

Role

Inquiries

Functionality Flow

Users browse interior designs and services

Users submit inquiries via contact form

Admin logs in to dashboard

Admin adds/edits interior projects and categories

Admin manages inquiries and site content

UI/UX Expectations

Clean, elegant, architecture-inspired design

Neutral color palette (white, beige, gray)

Grid-based layouts

Smooth hover effects

Mobile-first responsive design

Consistent typography and spacing

Deliverables

Fully functional Interior Design Website

Admin Dashboard with CRUD operations

REST API documentation

Clean, modular MERN folder structure

Responsive UI using Bootstrap & CSS

Optional Enhancements

Project filtering by category

Featured projects section

Image sliders for galleries

SEO-friendly meta tags

Soft delete for projects

Outcome

A professional MERN stack Interior Design Website suitable for real-world clients, portfolios, and resume projects, demonstrating frontend UI skills, backend API development, database design, and admin dashboard management.
